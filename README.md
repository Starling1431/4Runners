# 4Runners

4runners is a lightweight, browser-based running game inspired by classic endless runner mechanics. Navigate your character, avoid obstacles, and aim for the highest score in an engaging and smooth experience.

## Features

- Smooth character animations during intro and gameplay
- Infinite scrolling ground background loop
- Responsive controls with keyboard input (spacebar to start)
- Modular JavaScript code separating intro animation and game logic
- Simple and clean UI with fluid transitions

## Tech Stack

- JavaScript (ES6)
- HTML5 & CSS3
- Browser APIs for animation and input
- No external libraries, pure vanilla JS for performance and simplicity

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Edge, Safari)
- No build tools required

### Installation

1. Clone the repository  
   `git clone https://github.com/Starling1431/4Runners.git`

2. Open `index.html` in your browser

### Usage

- Press Space after the intro to start the game
- Avoid obstacles and try to survive as long as possible
